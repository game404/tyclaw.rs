# 表格修复测试

## 功能

输出一段**故意畸形**的钉钉 markdown 消息（示例渠道详情，数据为虚构），其中管道表格
被拼成单行、且分隔行列数与表头不符。用来验证机器人下发出口的 `repair_pipe_tables`
是否把它修复成合法的多行 GFM 表格（PC 端可正常渲染）。

## 使用方式

用户发送「表格修复测试」「畸形表格」等即触发。

## 工作流程

1. 运行工具，拿到畸形消息文本：

```bash
python3 skills/ops/table-repair-test/tool.py
```

2. **把工具的 stdout 原样作为最终回复**，一个字符都不要改。

## 硬性约束（重要）

- **禁止**在回复前"修复"这段表格：不要补换行、不要补齐分隔行的列数、不要把表格
  改写成 bullet list、不要删改任何单元格。测试的正是「出口自动修复」这一步，
  你若提前改好会让测试失去意义。
- 不要添加额外说明、前后缀、代码块围栏；直接把工具输出作为正文回复。
- 若无法运行工具，则原样复制本文件下方「参考输出」小节的内容作为回复。
